What the app does

Easy Pickleball Scoreboard keeps the official three-number pickleball call for doubles side-out scoring: serving score, receiving score, and server number.

Common questions

The opening server starts as server #2, so a new game opens at 0-0-2. Tap the team that won each rally; the app handles points, second server, side-out, game point, and win-by-two logic.

Rotate your phone during a game to show the larger landscape scoreboard. Rotate back to portrait for the full controls.
